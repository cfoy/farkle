# farkle

> A Vue.js project

## Prerequisites

This project requires:
- Node.js >= 18.0.0 (recommended: 20.18.1)
- npm >= 9.0.0

To manage Node.js versions, we recommend using [nvm](https://github.com/nvm-sh/nvm):

```bash
# Install the specified Node version
nvm install

# Use the specified Node version
nvm use
```

The `.nvmrc` file specifies the recommended Node.js version.

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests (uses ChromeHeadless)
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

## Testing

Unit tests run in headless Chrome via Karma. E2E tests use Nightwatch with ChromeDriver.

For detailed explanation on how things work, checkout the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
